## 📖 What is Celvra?

**Celvra** is a private, personal cloud platform built entirely with HTML, CSS, and JavaScript — no server, no backend, no tracking. It runs 100% in the browser and stores all data locally, giving every user complete control over their files, privacy, and experience.

Each person who registers gets their own isolated vault. Log in from any device, and your files and data follow you. Upload photos, videos, music, documents, contacts, and more — organize them into folders, share them with other platform users, and receive files with real-time notifications.

> Built with a bold animated dark interface that is entirely unique — not a template, not a theme. Designed from scratch.

## ✨ Features

### 🔐 Authentication
- Register with **username**, **email**, and **password**
- Each email address can only create **one account** — no duplicates
- **Remember Me** checkbox saves login credentials in the browser
- Session persists across page refreshes and device visits
- Password visibility toggle on all password fields
- Full logout with session clearing

### 📊 Dashboard
- Personalized greeting based on time of day
- Live file counts per section (Photos, Videos, Music, Contacts, Documents, Links)
- Unread notification counter
- Recent files feed showing your latest uploads

### 📸 Photos
- Upload **JPG, PNG, GIF, WEBP, SVG** files
- Create named folders, add files inside them
- Delete, Download, and Send buttons on every file
- Select multiple files and send them to another user at once

### 🎬 Videos
- Upload **MP4, MKV, AVI, MOV, WEBM** files
- Full folder support with custom names
- Per-file delete, download, and send controls

### 🎵 Music
- Upload **MP3, WAV, FLAC, AAC, OGG** files
- Organize tracks into named playlists/folders
- Full file management controls

### 📇 Contacts
- **Only accepts `.vcf` (vCard)** files — no other format allowed
- Designed for importing and storing contact backup files
- Send vCards to other platform users

### 📁 Documents
- Supports **PDF, DOCX, DOC, XLSX, XLS, TXT, APK, EXE** files
- Full folder organization system
- Download any document directly to your device

### 🔗 Important Links
- Save any URL with a custom label/name
- Open links in a new tab with one click
- Delete saved links anytime

### ✈️ File Transfer
- Send any file to **any registered Celvra user** by their email
- The recipient sees only your **username** — never your email address
- Real-time notification delivered to the recipient
- Recipient gets a **sound alert** and **pop-up notification** on any page
- Recipient accepts the file → it goes directly to the right section (photo → Photos, etc.)

### 🔔 Notification System
- Incoming file notifications appear in the **Notification Center**
- A **pop-up toast** appears anywhere on the platform with a "Visit Notification Center" button
- **Sound plays** when a notification arrives
- Badge counter on the sidebar and mobile topbar
- Mark files as received (accept) or dismiss
- Delete individual notifications or clear all at once

### 📂 Folder System
- Create folders in Photos, Videos, Music, and Documents
- Name each folder anything you like
- Add files directly inside folders
- Expand and collapse folders
- Delete folders (removes all contents)
- Files inside a folder do **not** appear outside it

### 📱 Responsive Design
- Fully functional on **mobile phones and tablets**
- Collapsible sidebar with hamburger menu on small screens
- Touch-friendly controls and tap targets
- Toast notifications adapt to mobile screen size

## 🎨 Design & Interface

Celvra uses a completely original animated dark interface:

- **Animated grid canvas** — subtle moving grid in the background
- **Floating orbs** — soft glowing color blobs that drift across the screen
- **Glassmorphism cards** — blurred, semi-transparent panels with border glow
- **Gradient accents** — teal (`#00f5c4`) and violet (`#7b5ea7`) throughout
- **Smooth page transitions** — fade + slide animations on every navigation
- **Pop-in modals** — spring-animated overlays for actions and alerts
- **Font: Outfit** — a geometric, modern display font (not overused)
- **Mono accents: JetBrains Mono** — for numbers, file sizes, and metadata
- **Custom scrollbar** — styled to match the dark theme
- **Micro-interactions** — hover lifts, glow on focus, scale on buttons

## 💾 How Data is Stored

Celvra uses the browser's `localStorage` API to store all user data — no server, no database, no cloud.

| What is stored | Where |
|---|---|
| User accounts (username, email, hashed data) | `localStorage` → `celvra_users` |
| Active session (logged-in user) | `localStorage` → `celvra_session` |
| Remember Me credentials | `localStorage` → `celvra_remember` |
| Files, folders, links, notifications | Inside each user's object in `celvra_users` |

> ⚠️ **Important:** Data is stored per browser. If you clear your browser's localStorage, data will be lost. For permanent cross-device syncing, consider integrating a backend like Firebase (future roadmap).

## 🔒 Privacy

- **Zero tracking** — no analytics, no cookies, no third-party scripts
- **No server** — your files never leave your device
- **Email privacy** — when you send a file to someone, they only see your **username**, never your email
- **One account per email** — prevents duplicate registrations
- **Session-based auth** — logging out clears your active session immediately

## 📬 File Transfer Flow

```
Sender selects file(s)
        ↓
Enters recipient's Celvra email
        ↓
Clicks "Send Now"
        ↓
File is written to recipient's notification queue in localStorage
        ↓
Recipient sees toast popup + hears notification sound
        ↓
"You received files from [Username]" modal appears
        ↓
Recipient clicks Accept → files saved to correct section automatically
```
